## 二、测试相册权限本地化

### 中文测试：
```bash
# 1. 确保系统语言为中文
设置 → 通用 → 语言与地区 → iPhone 语言 → 简体中文

# 2. 删除应用（如果已安装）
长按应用图标 → 移除 App

# 3. 重新运行
在 Xcode 中运行应用

# 4. 触发相册权限
在应用中选择照片

# ✅ 预期效果：
弹出提示："Feelm 需要访问您的相册以分析照片。"
```

### 英文测试：
```bash
# 1. 切换系统语言为英文
Settings → General → Language & Region → iPhone Language → English

# 2. 删除应用
Long press app icon → Remove App

# 3. 重新运行
Run in Xcode

# 4. 触发相册权限
Select photos in app

# ✅ 预期效果：
Popup shows: "Feelm needs access to your photo library to process analysis."
```

## 五、如果本地化没有生效

### 解决方案：

#### 1. 清理构建
```bash
# 在 Xcode 中
Product → Clean Build Folder
快捷键: Shift + Cmd + K
```

#### 2. 删除 DerivedData
```bash
rm -rf ~/Library/Developer/Xcode/DerivedData
```

#### 3. 重新安装应用
- 从设备/模拟器删除应用
- 重新运行 Xcode

#### 4. 重启 Xcode
```bash
# 关闭 Xcode
killall Xcode

# 重新打开
open Project_Color.xcodeproj
```

## 六、在 Xcode 中验证配置

### 检查本地化文件：
1. 打开 Xcode
2. 在项目导航器中查找 `InfoPlist.strings`
3. 应该看到它可以展开：
   ```
   📄 InfoPlist.strings
   ├── 🇺🇸 English
   └── 🇨🇳 Chinese (Simplified)
   ```

### 检查项目本地化设置：
1. 选中项目根节点 `Project_Color`
2. 点击 `Project`（不是 Target）
3. 在 **Info** 标签页中
4. 查看 **Localizations** 部分
5. 应该看到：
   - ✅ English
   - ✅ Chinese (Simplified)
